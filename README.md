# Sales Data Analysis Dashboard

## Project Overview

This project is an interactive **Sales Data Analysis Dashboard** built using Python and Streamlit.
It allows users to analyze sales data using filters, charts, and key performance indicators.

## Tools & Technologies

* Python
* Pandas
* Streamlit
* Plotly
* CSV Dataset

## Features

* Interactive Sales Dashboard
* KPI Metrics (Total Sales, Orders, Customers)
* Region, Category, and Date Filters
* Monthly Sales Trend Analysis
* Sales by Category Visualization
* Region-wise Sales Distribution
* Top 10 Products by Sales
* Download Filtered Data Option
* Interactive Data Table

## How It Works

1. The dataset is loaded using Pandas.
2. Sidebar filters allow users to select Region, Category, and Date range.
3. The dataset is filtered dynamically.
4. KPIs and charts update automatically based on filters.

## Dashboard Visualizations

* Line Chart: Monthly Sales Trend
* Bar Chart: Sales by Category
* Pie Chart: Sales by Region
* Bar Chart: Top 10 Products

## Project Outcome

This dashboard helps users quickly analyze sales performance and identify trends in different regions, categories, and products.
